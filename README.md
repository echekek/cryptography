# cryptography

substitution_cipher - Программа для зашифрования/расшифрования простым подстановочным шифром

affine_cipher - Программа для зашифрования/расшифрования аффинным шифром

affine_recurrent_cipher - Программа для зашифрования/расшифрования аффинным рекуррентным шифром

frequency_analysis - Программа для дешифрования частотным анализом 

cipher_breaker - Программа для взлома аффинного рекуррентного шифра

eas - реализация AES (зашифрование, расшифрование по ключу)
